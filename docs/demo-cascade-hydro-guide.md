# 梯级水电演示指南

演示模型：

- `cascade_hydro_dispatch`
- `cascade_hydro_dispatch_v1`

演示路径：

1. 首页查看“梯级水电优化调度”入口。
2. 进入模型资产中心查看模型演示说明。
3. 进入函数资产中心查看水位库容曲线、尾水位流量曲线和水电出力二维曲面。
4. 进入模型服务接口，使用样例参数在线调试。
5. 进入结果中心查看水电结果解释。
6. 使用 Agent 提问“这个模型为什么是 MILP？”。

能力边界：PWL 水电模型通过 HiGHS 求解；二维曲面三角剖分会增加 MILP 二进制变量规模。

二维出力曲面推荐策略：`triangulated_milp_exact`。

# NLP / Ipopt 演示指南

演示模型：`nonlinear_hydro_power_demo`。

核心关系：`power = k * flow * head`。

演示路径：

1. 首页查看“非线性水电出力 NLP 演示”入口。
2. Runtime 或首页检查 Ipopt 状态。
3. 模型资产中心查看 `nonlinear_hydro_power_demo` 的演示说明。
4. 模型服务接口使用样例参数在线调试。
5. 结果中心查看 NLP 结果解释。
6. Agent 提问“Ipopt 求解结果是不是全局最优？”。

能力边界：NLP / Ipopt 已支持真实求解；结果不承诺全局最优；Ipopt 不支持含整数变量的 MINLP。
